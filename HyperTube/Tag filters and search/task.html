Description

One of the big reasons you decided to create the site is convenience. To make it more convenient to use, you decide to add a video search so that every user can immediately find a video the are looking for. Let's implement that functionality and add a search and tag filter.

Objectives

Add a search form to your main page with one input element with the name q. The form should send GET requests to the same /tube/ page. The format of the result page looks similar but includes only the videos whose titles contain the input data.

Add another parameter tag to the /tube/ page to filter videos by the tag.

If you start the application on your computer with the command python manage.py runserver, the page at localhost:8000/tube/?q=ball should display videos whose titles contain the text "ball". The page at localhost:8000/tube/?tag=sport should display videos that have a tag "sport".
