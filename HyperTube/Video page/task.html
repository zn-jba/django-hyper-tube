Description

There is only one step left: last but not least! Make it possible to stream videos uploaded to your hosting.

Objectives

Create a new page for watching video streams. This page should contain the following:

  1. A link to the main page target="_blank" href="/tube/";
  2. A link to the login page target="_blank" href="/login/";
  3. A p tag with the title of the video;
  4. Links to the tube page with a filter for every tag, for example, <a target="_blank" href="/tube/?tag=sport">#sport</a>;
  5. A video tag and a source tag inside of it with a link to the file response and type, for example:
<video width="640" height="480" controls>
  <source src="/tube/football.mp4" type="video/mp4">
</video>

If you start the application on your computer with the command python manage.py runserver, the video from the example should be available at the address localhost:8000/tube/football.mp4. Its handler should use HttpResponse with the content_type of the video file and Accept-Ranges in headers with the bytes value.
