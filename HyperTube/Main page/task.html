Description

Now we are ready to give the video hosting a face, that is, a nice main page. This page will contain the name of your wonderful hosting, links to the uploaded videos, and some other basic things.

Objectives

The main goal of this stage is to create the main page that contains the following:

  1. An h2 header tag with the text Hypertube;
  2. A link to the login page target="_blank" href="/login/";
  3. The number of videos in the p tag;
  4. Links to the uploaded videos with titles;
  5. A link to the upload page target="_blank" href="/tube/upload/".

If you start the application on your computer with the command python manage.py runserver, the main page should be available at the address localhost:8000/tube/.
