Description

Your server is missing one of the most important features of any video hosting: the ability to upload new videos. After all the preparations have been made, it's time to add the ability to upload video files.

Objectives

Create a new page to add videos using a form. This form should send the data to the address /tube/upload/ with the POST method to create a new record. It should have two POST parameters title and tags (a string that has words separated by spaces) and one FILES parameter video. When the request is completed, the response should redirect you back to the /tube/ page.

If you start the application on your computer with the command python manage.py runserver, the page for uploading videos should be available at the address localhost:8000/tube/upload/. With the Python library requests, it will look like this:

requests.post('localhost:8000/tube/upload/', files={'video': your_video_data}, data={'title': 'My football training', tags='football sport'})

Video upload should be available for logged in users only.

Use the MEDIA_ROOT parameter from the settings of the project to store your files.

The page should also contain a link to the main page with target="_blank" href="/tube/".
